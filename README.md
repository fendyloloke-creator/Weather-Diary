# Weather Diary – Дневник погоды
**Автор:** Владимир Гусев 
## Описание программы
Weather Diary – это графическое приложение для ведения дневника погоды. Оно позволяет:
- Добавлять записи с указанием даты, температуры, описания и наличия осадков.
- Просматривать все записи в удобной таблице.
- Фильтровать записи по дате (точное совпадение) и по температуре (выше заданного порога).
- Сохранять все данные в JSON-файл и автоматически загружать их при запуске.
- Удалять ненужные записи.
## Как запустить приложение
1. Убедитесь, что установлен Python версии 3.6 или выше.
2. Скачайте файл `weather_diary.py` из репозитория.
3. Откройте терминал в папке с файлом и выполните:
   ```bash
   python weather_diary.py
